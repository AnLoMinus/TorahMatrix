## 🌟 מה זה TorahCraft?

**TorahCraft** היא מערכת למיפוי, סידור והנגשה של ידע אלוקי ורוחני, עם כלים אינטראקטיביים למורה ולתלמיד.  
היא שואפת להחזיר את הסדר העליון לעולם התחתון – באמצעות לימוד תורה המושתת על סודות הבריאה, וחיבורים בין דיסציפלינות.

🧬 חיבור בין:
- 📘 תורת הנגלה והנסתר
- 💡 תודעה פסיכולוגית־רוחנית
- 🔧 עקרונות מדעיים, אסטרטגיים, חינוכיים
- 🌀 חכמת הספירות, יסודות, פרד״ס
